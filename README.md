# 国男大冒险

这是一个基于HTML和JavaScript的互动式文字冒险游戏，玩家可以通过选择不同的选项来体验不同的剧情和结局。

## 项目结构

- `index.html`：游戏的主页面，包含基本的HTML结构和游戏逻辑。
- `data.json`：游戏数据文件，定义了游戏的标题、场景描述、选项和结局。
- `README.MD`：项目说明文件。

## 如何运行

1. 克隆或下载本项目到本地。
2. 确保项目文件在同一目录下。
3. 使用浏览器打开`index.html`文件即可开始游戏。

## 游戏玩法

1. 游戏开始时，玩家会看到一个场景描述。
2. 根据描述，玩家可以选择不同的选项。
3. 每个选项会引导玩家进入不同的场景或结局。
4. 游戏支持多次重玩，探索不同的剧情分支。
